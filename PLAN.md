# Albanian Speech-to-Text — Fine-Tuning Plan

## Goal
Fine-tune OpenAI's Whisper model on Albanian audio from a single speaker,
then use it to transcribe YouTube videos or audio files into text.
Model improves over time as new videos + transcripts are added weekly.

---

## Architecture

```
YouTube URL / audio file
        ↓
   yt-dlp (audio extraction)
        ↓
   audio chunking + forced alignment (WhisperX)
        ↓
   fine-tuned whisper-small (Albanian, speaker-specific)
        ↓
   plain text output
```

---

## Key Tools

| Tool | Purpose |
|------|---------|
| `yt-dlp` | Download audio from YouTube URLs |
| `ffmpeg` | Audio processing and format conversion |
| `whisper` (OpenAI) | The base speech-to-text model we fine-tune |
| `whisperX` (community) | Forced alignment — aligns your transcript to audio with timestamps |
| `transformers` (HuggingFace) | Fine-tuning framework |
| `pytorch` (MPS backend) | Runs on Apple Silicon (M3) |

---

## Incremental Training Workflow

Each week, retrain on ALL accumulated data (not just new files):

```
Week 1: 10 videos → fine-tune → model v1
Week 2: 20 videos total → retrain → model v2
Week 3: 30 videos total → retrain → model v3
```

---

## Steps

### Step 1 — Install tools
- Install: yt-dlp, ffmpeg, Python environment, WhisperX, openai-whisper
- Test: download audio from a YouTube video
- Homebrew version: 5.1.5

### Step 2 — Run base Whisper on Albanian audio
- Transcribe a sample video with the unmodified whisper-small model
- See current quality baseline before any fine-tuning

### Step 3 — Forced alignment
- Feed 1 video + your full transcript into WhisperX
- Output: JSON file with timestamped segments (auto-generated, no manual work)
- Test that the alignment looks correct

### Step 4 — Build training dataset
- Process all your videos + transcripts through the alignment pipeline
- Store as a HuggingFace dataset on disk
- Verify the segments look correct

### Step 5 — Fine-tune whisper-small
- Train on your prepared dataset (M3 MPS backend)
- First run: ~8-12 hours depending on data size
- Output: your custom model checkpoint

### Step 6 — Test fine-tuned model
- Transcribe the same sample from Step 2
- Compare output quality before vs. after fine-tuning

### Step 7 — Build the full pipeline script
- Single command: give it a YouTube URL or audio file
- It downloads, transcribes, and outputs Albanian text

---

## Data Format

**Input (you provide):**
- YouTube URL or downloaded audio file (mp3/wav/m4a)
- Full transcript as a `.txt` file (one big block of text, no timestamps needed)

**Output (auto-generated by pipeline):**
- Timestamped segments for training
- Final text file after transcription
